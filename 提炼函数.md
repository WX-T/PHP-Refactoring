
# 提炼函数1
原代码
------
```php
  $post_name = $_POST['name'];
  if(isset($post_name) && !empty($post_name)){
    ......
  }
  
  $post_age = $_POST['age'];
  if(isset($post_name) && !empty($post_name)){
    ......
  }
```
重构后
------
```php
  
  $post_name = $_POST['name'];
  if(isSetOrEmpty($post_name)){
    ......
  }
  $post_age = $_POST['age'];
  if(isSetOrEmpty($post_age)){
    ......
  }
   

  function isSetOrEmpty($value){
    if(isset($value) && !empty($value)){
      return true;
    }else{
      return false;
    }
  }
```
